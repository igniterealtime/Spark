## UI and Styling
The Spark is used mainly for work, so functionality is preferred over aesthetics.
However, people look on it for a whole day and it should not traumatize them by a bad design.
The color palette should be simple and consistent.
It should have a clean and simple interface that is easy to understand but not hide features.

### UI Design suggestions from the ChatGPT
Designing a chat UI people keep open all day (schools, clinics, small offices) is less about “looking cool” and more about visual calm, predictability, readability, and trust. Think closer to Slack (work-friendly) than WhatsApp (social-chat energy).

Below is a practical UI strategy you can implement directly.

1. Design goals for your audience

Your users:

teachers
administrators
clinic staff
receptionists
accountants
small teams

They need:

low eye fatigue 👀
clarity over style
neutral professionalism
slight friendliness (not corporate coldness)
predictable layout

So the UI should feel:

calm + trustworthy + lightweight + quietly friendly

Use this palette:

#F6F8F7

Very light warm-gray green tone
Feels softer than pure white.

Sidebar background
#E9F2EE

Muted green tint
Friendly but not childish

Primary accent (buttons, highlights)
#2F7D6B

Professional green-teal
Feels like:

stability
healthcare-safe
finance-safe
education-safe
Hover accent
#3F9A84
Message bubble (incoming)
#FFFFFF
Message bubble (outgoing)
#DFF3EA

Soft mint tone = calm + readable

Borders / separators
#D6E2DD

Thin separators reduce clutter fatigue

Text colors

Primary text:

#1F2A2A

Secondary text:

#6B7C7A

3. Why green works especially well here 🌿

Green communicates:

Industry	Interpretation
Healthcare	safety
Education	stability
Business	trust
Finance	balance
Messaging	calm activity

Blue is common — but green is less overused and feels warmer.

4. Layout structure that feels comfortable all day

Use a 3-panel layout:

| Sidebar | Chat list | Chat window |

Example:

Teams
Chats
Contacts
Settings

Sidebar width:

72px–96px

Chat list width:

260px–320px

Chat window:
flexible

This layout reduces mental switching fatigue.

5. Typography choice (VERY important)

Use one of these:

Best choices:

Inter
Roboto
Source Sans 3

Recommended sizes:

Chat text:

15px–16px

Sidebar:

14px

Headers:

18px–20px

Line spacing:

1.4–1.5

Comfortable reading all day 📖

6. Message bubble design rules

Avoid rounded cartoon bubbles ❌

Use:

border-radius: 8px

not

border-radius: 20px

Looks more professional.

7. Use gradients very subtly

Example sidebar gradient:

linear-gradient(
180deg,
#EAF5F1,
#E3F0EB
)

Almost invisible = feels premium without distraction ✨

8. Notification color strategy (critical)

Never use bright red everywhere.

Instead:

Unread badge:

#5FAF95

Mention badge:

#F2A65A

Error:

#D96C6C

Success:

#6FBF8F

Muted notifications reduce stress during long workdays.

9. Micro-interactions that increase comfort

Add:

✔ soft hover fades
✔ subtle message appear animation
✔ typing indicator
✔ read receipt dots
✔ smooth scroll inertia

Animation duration:

120ms – 180ms

Feels responsive but calm.

10. Dark mode (very important for all-day apps)

Use this palette:

Background:

#1E2423

Sidebar:

#18201E

Outgoing message:

#244C41

Incoming message:

#2A3231

Accent:

#5FAF95

Dark mode dramatically improves usability for clinics & admins working late 🌙

11. Personality level of the UI

Target tone:

70% professional
20% friendly
10% modern

Avoid:

❌ playful emojis in UI
❌ bright gradients
❌ strong blues
❌ neon accents

Users should forget the interface exists.

12. Extra trust signals that matter for schools & clinics

Add:

🔒 lock icon near messages
🕓 visible timestamps
👤 clear sender identity
📎 predictable attachment behavior

These subconsciously increase confidence.
